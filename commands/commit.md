description: Realiza o fluxo completo de commit com Gitmoji: commits agrupados por contexto, push e PR.
---

Execute o fluxo completo de commit conforme a skill git-commit:
1. Inspecione `git status` e `git diff`
2. Se estiver na `main`, crie uma branch descritiva
3. Agrupe mudanças por contexto/domínio
4. Gere commits no padrão Gitmoji em português no imperativo
5. Faça push para o remoto
6. Abra PR com título claro e descrição organizada
